!!!Struts标签
The Struts 2 Tags can be divided into two types: __Generic Tags__ and __UI Tags__ .

!!Generic Tags
Generic tags are used for controlling the execution flow when the pages render.\\
Generic tags are classified into Control Tags and Data Tags.
%%tabbedSection 
%%tab-ControlTags
The Control Tags are used for flow control, such as if, else and iterate.
* [if|Struts.Tags.if]
* [elseif|Struts.Tags.elseif]
* [else|Struts.Tags.else]
* [append|Struts.Tags.append]
* [generator|Struts.Tags.generator]
* [iterator|Struts.Tags.iterator]
* [merge|Struts.Tags.merge]
* [sort|Struts.Tags.sort]
* [subset|Struts.Tags.subset]
/%
%%tab-DataTags
The Data Tags are used for data manipulation or creation, such as bean, push, and i18n.
* [a|Struts.Tags.a]
* [action|Struts.Tags.action]
* [bean|Struts.Tags.bean]
* [date|Struts.Tags.date]
* [debug|Struts.Tags.debug]
* [i18n|Struts.Tags.i18n]
* [include|Struts.Tags.include]
* [param|Struts.Tags.param]
* [property|Struts.Tags.property]
* [push|Struts.Tags.push]
* [set|Struts.Tags.set]
* [text|Struts.Tags.text]
* [url|Struts.Tags.url]
/%
/%

!!UI Tags
UI Tags display data in rich and reusable HTML.\\
UI Tags are of two types __Form Tags__ and __Non-Form Tags__ .
